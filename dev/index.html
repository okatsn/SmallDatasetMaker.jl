SmallDatasetMaker.SourceData — Method

SourceData(srcfile, packagename, datasetname, title, zipfile, rows, columns, description, timestamps)

srcfile is the path to the source file, the package_name will be the folder that the file resides, the dataset_name will be the name of the data without extension.

If timestamps not specified, it will be today().

SmallDatasetMaker.SourceData — Method

SourceData(row::DataFrameRow) returns

SourceData(
    abspath(row.RawData),    # srcfile::Union{Missing,String}
    row.PackageName,         # package_name::String
    row.Dataset,             # dataset_name::String
    row.Title,               # title::Union{Missing,String}
    abspath(row.ZippedData), # zipfile::String
    row.Rows,                # rows::Int
    row.Columns,             # columns::Int
    row.Description,         # description::Union{Missing,String}
    row.TimeStamp,           # timestamps::TimeType
    )

SmallDatasetMaker.dataset_table — Method

dataset_table().

The path to the index table for datasets in SmallDatasetMaker. If SmallDatasetMaker is added using pkg> dev SmallDatasetMaker in other project/environment, dataset_table() returns "~/.julia/dev/SmallDatasetMaker/src/../data/doc/datasets.csv".

The reason for dataset_table to be a function rather than a constant is that I can redefine it in the scope of test. See test/compdecomp.jl.

SmallDatasetMaker.datasets — Method

datasets() returns the table of this dataset, and define the global variable SmallDatasetMaker.__datasets as this table.

SmallDatasetMaker.get_package_dataset_name — Method

Given path to the source file, get_package_dataset_name(srcpath) derive package name and dataset name from the srcpath.
